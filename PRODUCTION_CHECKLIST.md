# Production Checklist

- [ ] Install dependencies with `npm install`
- [ ] Build with `npm run build`
- [ ] Verify route navigation and browser refresh behavior
- [ ] Verify static assets load correctly
- [ ] Validate dashboard, analytics, alarms, automation, topology, reports, access control, and settings modules
- [ ] Confirm deployment settings for hosting target (Vercel, Netlify, or DigitalOcean)
