# Open Valley Production Checklist

## Build and Packaging
- [ ] Run `npm install`
- [ ] Run `npm run build`
- [ ] Confirm build output exists in `dist/`

- [ ] Application loads successfully
- [ ] Route navigation and browser refresh behavior are correct
- [ ] Static assets load without errors

- [ ] Vercel settings validated

- [ ] DigitalOcean App Platform settings validated
