# Open Valley Development Guide

## Prerequisites
- Node.js 20+
- npm
- Git

## Install and Run
```bash
npm install
npm run dev
```

Local URL: `http://localhost:5173`

## Build Validation
Before merging or releasing, run:

```bash
npm run build
```

## Recommended Workflow
1. Install dependencies.
2. Start local development server.
3. Implement and validate changes.
4. Run a production build before deployment.
